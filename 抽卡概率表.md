## 概率计算公式总结

### 计算步骤（按顺序执行）

1. **基础概率**：2星45%，3星30%，4星15%，5星10%

2. **加算乘区（王的运势）**：
   - 如果拥有"王的运势"状态，每个老婆为4星和5星概率+2%
   - 4星概率 = 15% + (老婆数量 × 2%)
   - 5星概率 = 10% + (老婆数量 × 2%)

3. **幸运E处理（特殊乘区）**：
   - 如果拥有"幸运E"状态，所有品质概率先减半，但不低于20%
   - 2星：45% × 0.5 = 22.5%
   - 3星：max(20%, 30% × 0.5) = 20%
   - 4星：max(20%, 4星当前概率 × 0.5)
   - 5星：max(20%, 5星当前概率 × 0.5)
   - **注意**：如果先应用了王的运势，幸运E会基于调整后的概率计算

4. **运势倍数计算（最终乘区）**：
   - 如果有吉星如意且老婆是吉星：倍数 = 1.30（覆盖运势）
   - 否则：倍数 = 1.0 + (stars - 4) × 0.05
     - 大吉(7星)：1.15
     - 吉(6星)：1.10
     - 小吉(5星)：1.05
     - 中平(4星)：1.00
     - 小凶(3星)：0.95
     - 凶(2星)：0.90
     - 大凶(1星)：0.85

5. **运势影响**：只影响4~5星
   - 4星概率 = 4星当前概率 × 运势倍数
   - 5星概率 = 5星当前概率 × 运势倍数

6. **归一化**：所有概率总和归一化为100%

### 计算顺序示例（中平 + 幸运E + 王的运势（3个老婆））

1. 基础：2星45%，3星30%，4星15%，5星10%
2. 加算（王的运势）：2星45%，3星30%，4星21%（15%+6%），5星16%（10%+6%）
3. 幸运E减半：2星22.5%，3星20%（max(20%, 30%×0.5)），4星20%（max(20%, 21%×0.5)），5星20%（max(20%, 16%×0.5)）
4. 运势（中平1.00）：2星22.5%，3星20%，4星20%，5星20%
5. 归一化：2星27.27%，3星24.24%，4星24.24%，5星24.24%

**注意**：当同时拥有幸运E和王的运势时，王的运势会被幸运E的"不低于20%"限制抵消，因为加算后的概率被减半后又被限制回20%。

## 注意事项

1. **王的运势优先级**：王的运势是加算加成，在概率计算的第一步应用（加算乘区）
2. **幸运E优先级**：幸运E先于运势计算，先减半再应用运势倍数
3. **吉星如意优先级**：如果吉星如意生效，完全覆盖运势计算，固定使用1.30倍数
4. **归一化**：所有概率计算完成后会归一化，确保总和为100%
5. **2~3星不受运势影响**：只有4~5星会受到运势和吉星如意的影响
6. **王的运势与幸运E的交互**：
   - 王的运势是加算，在第一步应用
   - 幸运E是乘算，在第三步应用
   - 如果同时拥有，计算顺序为：基础 → 加算（王的运势） → 幸运E减半
   - **重要**：由于幸运E有"不低于20%"的限制，当王的运势加算后的概率被幸运E减半后，如果结果低于20%会被限制回20%，导致王的运势加成被抵消
   - 例如：4星基础15% + 王的运势6% = 21%，然后幸运E减半 = 10.5%，但被限制为20%，所以最终为20%（王的运势失效）
