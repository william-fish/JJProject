# 群二次元老婆插件道具卡说明

本文档整理了 `astrbot_plugin_animewifex` 插件当前可用的 42 种道具卡及其效果，并补充了常用指令说明和运势系统说明，方便快速查询。
**通用规则：任何道具提升的概率值最高封顶为 90%。**

## 道具卡列表

① **牛魔王**  
- 获得"牛魔王"状态。  
- 状态效果：今日你使用「牛老婆」成功率 +30%；今日别人对你使用「牛老婆」成功率 +30%；今日你可以无视除「贤者时间」外的牛老婆禁用效果；今日你从其他渠道获得的「牛老婆」次数翻倍。

② **开后宫**  
- 获得"开后宫"状态。  
- 状态效果：今日「抽老婆」次数等于「换老婆」次数（含额外次数）；可同时拥有多个老婆，但拥有 ≥2 个老婆时会以 `5%*1.33^老婆数量` 概率触发修罗场（处于露出状态时必定触发修罗场），失去全部老婆，触发修罗场后获得一次抽老婆（换老婆）机会；今日无法使用「换老婆」「重置牛」「重置换」；牛成功后会追加一次抽老婆流程（不消耗次数）。

③ **贤者时间**  
- 获得"贤者时间"状态。  
- 状态效果：清空你今日已有的所有道具效果并禁止继续使用道具；今日完全免疫一切道具或群体效果（包括他人对你施放的道具）。

④ **开impart**  
- 将今日拥有老婆的所有成员随机重新分配老婆。  
- 开后宫成员会保持原有老婆数量。  
- 自动取消相关的交换请求。

⑤ **纯爱战士**  
- 获得"纯爱战士"状态。  
- 状态效果：今日你的老婆不会被「牛老婆」夺走；你无法使用「换老婆」「牛老婆」「交换老婆」「同意交换」，他人也不能对你使用后两项；若在特殊效果下依旧成功夺得老婆，将触发提示"坚守纯爱的你拒绝了牛来的老婆，不要违背自己的内心哦"并放弃该老婆。

⑥ **雌堕**（需 @ 目标）  
- 50% 概率让目标成为你的老婆（头像入库）。  
- 50% 概率你成为目标的老婆（自己的头像入库）。 

⑦ **雄竞**（需 @ 目标）  
- 获得"雄竞"状态。  
- 状态效果：今日你「抽老婆」有 35% 概率抽到与目标相同的老婆。

⑧ **苦主**

⑨ **黄毛**

## 运势系统

每个用户每天会获得一次运势，运势类型包括：**大吉**（7颗星）、**吉**（6颗星）、**小吉**（5颗星）、**中平**（4颗星）、**小凶**（3颗星）、**凶**（2颗星）、**大凶**（1颗星）。

### 运势生成规则
- 每个用户的运势每天生成一次，生成后今日运势将不会变。
- 生成时机：用户今日第一次使用了老婆插件的任何指令时。
- 使用 `今日运势` 指令可以查看今日运势。

### 运势影响
- 以4颗星（中平）为基准，每多/少一颗星，用户今日的所有概率将乘以 1±0.05（每颗星0.05）。
- 例如：7颗星（大吉）= 概率 × 1.15；1颗星（大凶）= 概率 × 0.85。
- 该计算位于在所有计算之后，为最终概率算法（拥有幸运E的除外）。

### 运势内容
- 占卜今日运势，运势会影响你今日的概率事件。

## 常用命令速览

- `查状态`：查看自己当前所拥有的全部效果、额外次数及群加成。  
- `老婆插件帮助`：列出所有可用指令与道具名称，便于快速检索。  
- `查道具`：查看今日尚未使用的道具卡列表。  
- `今日运势`：查看今日运势，包括运势类型、星级、谚语、今日吉星、宜忌建议等。  
- `重置盲盒`：按规则重置盲盒，重新抽取可用道具（需满足使用限制）。  
- `赠送 / 索取 / 同意赠送 / 同意索取 / 查看道具请求`：管理当日的道具互赠流程，每位用户每天最多发起 2 次赠送和 2 次索取。
