# 群二次元老婆插件道具卡说明

本文档整理了 `astrbot_plugin_animewifex` 插件当前可用的 15 种道具卡及其效果，方便快速查询。

## 道具卡列表

① **牛魔王**  
- 今日你使用「牛老婆」成功率 +30%。  
- 今日别人对你使用「牛老婆」成功率 +30%。  
- 无视除「贤者时间」外使自己无法使用「牛老婆」的效果。

② **开后宫**  
- 今日「抽老婆」次数等于「换老婆」次数（含额外次数）。  
- 可同时拥有多个老婆，但拥有 ≥2 个老婆时会以 `(老婆数量-1)*10%` 概率触发修罗场，失去全部老婆。  
- 今日无法使用「换老婆」「重置牛」「重置换」。  
- 牛成功后会追加一次抽老婆流程。

③ **贤者时间**  
- 清空你今日已有的所有道具效果并禁止继续使用道具。  
- 今日完全免疫一切道具或群体效果（包括他人对你施放的道具）。

④ **开 impart**  
- 将今日拥有老婆的所有成员（除贤者时间者）随机重新分配老婆。  
- 开后宫成员会保持原有老婆数量。  
- 自动取消相关的交换请求。

⑤ **纯爱战士**  
- 今日你的老婆不会被「牛老婆」夺走。  
- 你无法使用「换老婆」「牛老婆」「交换老婆」「同意交换」，他人也不能对你使用后两项。

⑥ **雌堕**（需 @ 目标）  
- 50% 概率让目标成为你的老婆（头像入库）。  
- 50% 概率你成为目标的老婆（自己的头像入库）。  
- 目标处于「贤者时间」则无效。

⑦ **雄竞**（需 @ 目标）  
- 今日你「抽老婆」有 30% 概率抽到与目标相同的老婆。

⑧ **苦主**  
- 今日被「牛老婆」必定成功，但每次被牛都会额外获得 1 次「换老婆」次数。  
- 禁止使用「拒绝交换」。

⑨ **黄毛**  
- 将今日已使用的「换老婆」次数累加到「牛老婆」次数。  
- 下次「牛老婆」必定成功。  
- 今日无法再使用「换老婆」。

⑩ **何意味**  
- 随机触发其中一个效果：  
  - 被禁言 300 秒。  
  - 「换老婆」或「牛老婆」次数翻倍 / 归零（随机其一）。  
  - 今日「换老婆」50% 免消耗或 50% 失败（随机其一）。  
  - 再次触发一次「开 impart」。  
  - 立刻执行一次「换老婆」（不消耗次数）。  
  - 随机触发两种额外效果。  
  - 获得 1 张随机道具卡或 2 张随机道具卡。  
  - 获得一次额外「抽盲盒」机会。  
  - 今日禁止使用道具。  
  - 失去所有当日未使用道具。  
  - 强制使用一张自己持有的非指向类道具卡。

⑪ **白月光**  
- 今日增加 1 次「选老婆」次数。

⑫ **公交车**  
- 今日你对他人使用「交换老婆」无需对方同意，直接强制交换。  
- 今日无法使用「牛老婆」（若有牛魔王则可豁免禁用）。

⑬ **病娇**  
- 今日你的老婆不会被「牛老婆」夺走。  
- 当你已有老婆且牛成功时，会随机触发特殊事件（杀掉新老婆 / 自己被禁言 / 老婆自杀 / 获得「何意味」）。  
- 使用「打老婆」时不会触发失去全部老婆的惩罚，并附带 10 条专属愉悦台词。

⑭ **儒夫**  
- 今日获得 10 次「打老婆」使用次数。

⑮ **熊出没**  
- 今日可无限次使用「勾引」，但每次有 25% 概率被禁言 120 秒。  
- 将「勾引」次数设置为无限（内部记为 -1）。

---

如需查看当日持有的道具卡，请使用命令 `查道具`。如果想重新抽取，请根据权限或管理员操作使用 `重置盲盒`。更多指令细节请参考插件代码。*** End Patch
