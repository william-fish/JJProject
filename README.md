# 群二次元老婆插件道具卡说明

本文档整理了 `astrbot_plugin_animewifex` 插件当前可用的 28 种道具卡及其效果，并补充了常用指令说明，方便快速查询。
**通用规则：任何道具提升的概率值最高封顶为 90%。**

## 道具卡列表

① **牛魔王**  
- 今日你使用「牛老婆」成功率 +30%。  
- 今日别人对你使用「牛老婆」成功率 +30%。  
- 今日你可以无视除「贤者时间」外的牛老婆禁用效果。  
- 今日你从其他渠道获得的「牛老婆」次数翻倍。

② **开后宫**  
- 今日「抽老婆」次数等于「换老婆」次数（含额外次数）。  
- 可同时拥有多个老婆，但拥有 ≥2 个老婆时会以 `5%*1.35^老婆数量` 概率触发修罗场，失去全部老婆。  
- 今日无法使用「换老婆」「重置牛」「重置换」。  
- 牛成功后会追加一次抽老婆流程。

③ **贤者时间**  
- 清空你今日已有的所有道具效果并禁止继续使用道具。  
- 今日完全免疫一切道具或群体效果（包括他人对你施放的道具）。

④ **开impart**  
- 将今日拥有老婆的所有成员（除贤者时间者）随机重新分配老婆。  
- 开后宫成员会保持原有老婆数量。  
- 自动取消相关的交换请求。

⑤ **纯爱战士**  
- 今日你的老婆不会被「牛老婆」夺走。  
- 你无法使用「换老婆」「牛老婆」「交换老婆」「同意交换」，他人也不能对你使用后两项。  
- 若在特殊效果下依旧成功夺得老婆，将触发提示“坚守纯爱的你拒绝了牛来的老婆，不要违背自己的内心哦”并放弃该老婆。

⑥ **雌堕**（需 @ 目标）  
- 50% 概率让目标成为你的老婆（头像入库）。  
- 50% 概率你成为目标的老婆（自己的头像入库）。  
- 目标处于「贤者时间」则无效。

⑦ **雄竞**（需 @ 目标）  
- 今日你「抽老婆」有 35% 概率抽到与目标相同的老婆。

⑧ **苦主**  
- 今日被「牛老婆」必定成功，每次失去老婆都会额外获得 1 次「换老婆」次数。  
- 禁止使用「拒绝交换」。

⑨ **黄毛**  
- 将今日已使用的「换老婆」次数累加到「牛老婆」次数。  
- 下次「牛老婆」必定成功。  
- 今日无法再使用「换老婆」。

⑩ **何意味**  
- 随机触发其中一个效果：  
  - 被禁言 300 秒。  
  - 「换老婆」或「牛老婆」次数翻倍 / 归零（随机其一）。  
  - 今日「换老婆」50% 免消耗或 50% 失败（随机其一）。  
  - 再次触发一次「开 impart」。  
  - 立刻执行一次「换老婆」（不消耗次数）。  
  - 随机触发两种额外效果。  
  - 获得 1 张随机道具卡或 2 张随机道具卡。  
  - 获得一次额外「抽盲盒」机会。  
  - 今日禁止使用道具。  
  - 失去所有当日未使用道具。  
  - 强制使用一张自己持有的非指向类道具卡。

⑪ **白月光**  
- 今日增加 1 次「选老婆」次数。

⑫ **公交车**  
- 今日你对他人使用「交换老婆」无需对方同意，直接强制交换。  
- 今日无法使用「牛老婆」（若有牛魔王则可豁免禁用）。

⑬ **病娇**  
- 今日你的老婆不会被「牛老婆」夺走。  
- 当你已有老婆且牛成功时，会随机触发特殊事件（杀掉新老婆 / 自己被禁言 / 老婆自杀 / 获得「何意味」）。  
- 使用「打老婆」时不会触发失去全部老婆的惩罚，并附带 10 条专属愉悦台词。

⑭ **儒夫**  
- 今日获得 10 次「打老婆」使用次数。

⑮ **熊出没**  
- 今日可无限次使用「勾引」，但每次有 25% 概率被禁言 120 秒。  
- 将「勾引」次数设置为无限（内部记为 -1）。

⑯ **宝刀未老**  
- 仅当今日「牛老婆」次数为0时可发动。  
- 立即获得 4 次额外的「牛老婆」使用次数。

⑰ **鹿鹿时间到了**  
- 为本群所有成员增加 1 次「换老婆」使用次数。  
- 群加成会实时体现在每位成员的换老婆上限中。

⑱ **开明盒**  
- 使用格式：`使用开明盒 道具名`，可自选并立即获得任意一张现有道具卡。  
- 可与其它道具效果叠加，抽到的自选卡需照常在当日内使用。

⑲ **龙王**  
- 当日你的老婆至少被「牛」走1次老婆时可使用，否则提示“隐忍，还未到使用的时候……”。  
- 使用后立即获得开后宫效果，先前被牛走的所有老婆都会回到你名下，并清除相关损失记录。

⑳ **二度寝**  
- 赋予"二度寝"状态：你下次使用的道具卡数值效果翻倍（包括正向与负向，例如指令次数、成功率、失败概率、禁言时间等）。  
- 使用后的下一张道具若未生效，翻倍状态会保留至生效为止。

㉑ **烧火棍**  
- 清空你今日的所有数据（效果、道具、老婆、记录等）。  
- 获得"棍勇"状态：获得 1 次额外的"重置"次数和 1 次额外的"重置盲盒"机会。  
- 在抽老婆和换老婆时有 35% 概率抽到关键词包含「回复术士的重启人生」的角色。

㉒ **未来日记**  
- 你下次的「抽老婆」或「换老婆」将会换到关键字为「我妻由乃」的角色。  
- 并且获得「病娇」效果。

㉓ **爱马仕**  
- 获得"爱马仕"状态。  
- 今天你抽老婆或换老婆只会抽到关键字包含「赛马娘」的角色。

㉔ **帕青哥**  
- 清空你所有道具。  
- 随机触发一种效果：  
  - 用完今日所有的换老婆、牛老婆、重置、重置盲盒次数，随机获得 1~6 次换老婆的次数、1~6 次牛老婆的次数、0~3 次重置次数、0~3 次重置盲盒次数。  
  - 获得"777"效果：获得 1 次额外抽盲盒次数，且今日抽盲盒必定触发一次暴击。  
  - 随机获得三个状态（flags）效果（不能重复获得已有的状态）。  
  - 被禁言 300 秒。

㉕ **苦命鸳鸯** 
- 随机锁定当前群内一位拥有老婆的群友（不含自己）。  
- 你与对方会失去所有老婆，并互相成为对方的新老婆（头像入库）。

㉖ **牛道具**（需 @ 目标）  
- 随机从目标今日持有的道具中夺取 0~2 张，实际数量视对方库存而定，可能一无所获。

㉗ **偷拍**（需 @ 目标）  
- 你会先失去当日全部老婆，再复制目标当前的老婆阵容。  
- 若自身未处于开后宫状态，则只会获得目标名单中的第一位。

㉘ **复读**（需 @ 目标）  
- 你会先失去当日全部道具卡，再复制目标当前的道具清单。  
- 目标处于贤者时间或没有道具时使用无效。

## 常用命令速览

- `查状态`：查看自己当前所拥有的全部效果、额外次数及群加成。  
- `老婆插件帮助`：列出所有可用指令与道具名称，便于快速检索。  
- `查道具`：查看今日尚未使用的道具卡列表。  
- `重置盲盒`：按规则重置盲盒，重新抽取可用道具（需满足使用限制）。

更多指令细节请参考插件代码或在群内试用。*** End Patch
